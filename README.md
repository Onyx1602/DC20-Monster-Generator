# DC20-Monster-Generator
![GitHub all releases](https://img.shields.io/github/downloads/Onyx1602/DC20-Monster-Generator/total.svg) [![discord members](https://discord-live-members-count-badge.vercel.app/api/discord-members?guildId=1346272545195950151)](https://discord.gg/onyxstudios)

The DC20 Monster Generator helps Game Masters quickly create balanced monsters for their games in Foundry VTT. Features:

Monster Generation Logic: Automatically calculates HP, PD, AD, and damage based on level, size, and role. Creature size adjusts defenses, and roles further modify stats for balance.,
Simple Setup: Install directly from the Foundry VTT module browser. Fully compatible with Foundry VTT v13.,
Customizable Creation: Generate monsters by name, size, type, level, and role. Upload your own monster images.,
Ability Assignment: Monsters receive role- and type-based abilities automatically, with the option to reroll for new results. Just press the on demand macro to select an ability to regenerate.,
Dynamic Scaling: Buff or debuff monsters to higher or lower levels with a using the table provided by the generationâ€”ideal for reusing the same creature across tiers of play.,
Summoner Role Support: Automatically generates linked summons with predefined stats and abilities, which scale with items and level.,
DCCrit Importer, Import monsters from DCCrit to FoundryVTT.

Magic Generator make making magic items 10 times easier.
Magic Item Buiilder have ideas for magic items choose what those properties are with this.

This free tool makes monster prep fast, flexible, and scalable for DC20 GMs.
